What is an Observer?

An Observer consumes the values that an Observable produces.

An Observer is a set of 3 callbacks, one for each type of notification that an Observable can produce.

Example:

      
this.trades.getTrades()
  // Emits a value after 10 secs, subscription completes.
  .takeUntil(Observable.timer(10000))
  .subscribe(
    // 3 different callbacks for the different values the Observable can deliver.
    data => 
      this.observerData.unshift(`${{ '{' }}data[0].price{{ '}' }}} 
      I am the most recent ${{ '{' }}data[0].side === 'buy' ? 'BUY' : 'SELL'{{ '}' }} price of ETH/USD`),
    err => this.observerData.unshift('error occurred!'),
    () => this.observerData.unshift('complete after 10 secs!')
  );
